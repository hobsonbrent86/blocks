# Security

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security problems.

Email the maintainers privately with:

- Description of the issue
- Steps to reproduce
- Impact assessment (if known)

We will acknowledge within 72 hours and work on a fix before any public disclosure.

## Scope

This repo covers the Blocks CLI, verify engine, MCP server, and reference examples. It does **not** run your production applications — blocks projects execute in your environment with your credentials.
